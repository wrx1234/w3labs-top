# W³ Labs 官网

## 运行

```bash
npm install
npm run dev
```

打开 http://localhost:3000

## 已实现

- 首页: Banner / NavBar / Hero / Stats / Mission / Content / Services / Cases / Partners / CTA / Footer
- `/events` 子页: 活动介绍 / Agenda / 报名表单
- 全局中英文切换
- 明暗主题切换
- 浮动快捷侧栏
- Next.js 14 + Tailwind + TypeScript + shadcn 风格基础组件

## 验证

```bash
npm run lint
npm run build
```
